# Lip Sync Avatar - Real-time 3D Avatar with Audio Lip Synchronization

A React Three Fiber project that creates realistic lip-sync animations for 3D avatars using audio files and phoneme data generated by Rhubarb Lip Sync.

Live Demo : https://lip-sync-three-fiber.netlify.app/

This project demonstrates real-time lip synchronization for 3D avatars by mapping audio phonemes to facial morph targets. The avatar's mouth movements are synchronized with speech audio, creating a realistic talking animation.
