# Lip Sync Avatar - Real-time 3D Avatar with Audio Lip Synchronization

A React Three Fiber project that creates realistic lip-sync animations for 3D avatars using audio files and phoneme data generated by Rhubarb Lip Sync.

## ðŸŽ¯ Overview

This project demonstrates real-time lip synchronization for 3D avatars by mapping audio phonemes to facial morph targets. The avatar's mouth movements are synchronized with speech audio, creating a realistic talking animation.

## âœ¨ Features

- **Real-time Lip Sync**: Avatar mouth movements synchronized with audio playback
- **Oculus Visemes Support**: High-quality facial animations using Oculus viseme morph targets
- **Smooth Transitions**: Configurable morph target smoothing for natural animations
- **Ready Player Me Integration**: Compatible with Ready Player Me avatar models
- **Interactive Controls**: Real-time controls using Leva for testing and tweaking

### Prerequisites

- Rhubarb Lip Sync (included in `Rhubarb-Lip-Sync-1.14.0-Windows/`)

### Installation

```bash

# Start development server
npm run dev
```
